  ### Happy Project <img src="https://github.com/DarlonHenrique/happy-project-nlw-3/blob/main/public/images/logo-icon.png" alt="happy icon" width="30px"/>
  ![happy image](https://github.com/DarlonHenrique/happy-project-nlw-3/blob/main/.github/Happy.png)
  ![happy gif demo](https://github.com/DarlonHenrique/happy/blob/main/.github/happy-demo.gif)
  the happy app was developed with the intention of helping people book visits to social assistance houses and making the day of several people better
  ### technologies used
in this app I used the **Handlebars** <img src="https://cdn.worldvectorlogo.com/logos/handlebars.svg" alt="handlebars icon" width="20px" height="20px"/>  to create the project template

I used **SQlite** <img src="https://cdn.worldvectorlogo.com/logos/sqlite.svg" alt="sqlite icon" width="20px" height="20px"/> as DB

and used the **Nodemon** <img src="https://cdn.worldvectorlogo.com/logos/nodemon.svg" alt="nodemon icon" width="20px" height="20px"/> for fast refresh and server
the style was done with pure **CSS** <img src="https://cdn.worldvectorlogo.com/logos/css-5.svg" alt="CSS icon" width="20px" height="20px"/>

I implemented the **OpenStreetMap API** <img src="https://upload.wikimedia.org/wikipedia/commons/thumb/b/b0/Openstreetmap_logo.svg/1024px-Openstreetmap_logo.svg.png" alt="open street api icon" width="20px" height="20px"/> to generate a map and other features

and pure **javascript** <img src="https://cdn.worldvectorlogo.com/logos/logo-javascript.svg" alt="javascript icon" width="20px" height="20px"/> to create the back-end.
